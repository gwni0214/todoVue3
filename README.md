# Vue Todo List 만들기  

    구현해야 할 기능
    1. todo 입력
        a. 토스트 팝업 구현

    2. todo 삭제하기
        a. 삭제에 대한 확인 Modal 구현

    3. todo 수정하기
        a. 로컬스토리지 JSON 활용

    4. 리스트 체크하기 기능
